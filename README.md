# 도형놀이 (ShapePlay)

격자 위에서 도형을 직접 만들고, **밀기·뒤집기·돌리기**를 3D 입체 애니메이션으로 눈으로 보며 공간 지각력을 키우는 초등 수학 교육용 웹앱.

> 초등 4학년 1학기 '도형의 이동' 단원을 위해 만들어졌습니다.

---

## 주요 기능

- **도형 만들기** — 격자 위 꼭짓점을 클릭해 자유롭게 다각형 생성
- **도형 이동하기** — 밀기(평행이동) · 뒤집기(대칭) · 돌리기(회전) 3단계 조작 모달
- **3D 플립 애니메이션** — 뒤집기/돌리기 시 종이가 넘어가는 입체 궤적 렌더링
- **타임라인 되돌리기** — ← / → 버튼으로 조작 기록 앞뒤로 복기
- **줌 인/아웃** — 캔버스 중심 기준 확대·축소
- **다크모드 자동 지원** — OS 시스템 설정에 따라 자동 전환
- **미션 모드** — 흐릿한 목표 그림자에 도형을 딱 맞추는 게임 모드 *(선택 기능)*

|------|------|

> 서버·DB 없이 브라우저 100% 로컬 구동 — 유지비 0원

## 시작하기

```bash
# 패키지 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 사용 방법

1. **도형 만들기** 버튼 클릭 → 격자 위 꼭짓점을 순서대로 클릭 → 첫 점 근처를 클릭하면 도형 완성
2. **도형 이동하기** 버튼 클릭 → 밀기·뒤집기·돌리기 조건 입력 → 3D 애니메이션으로 이동 확인
3. **← / →** 버튼으로 과거 상태로 되돌리거나 다시 앞으로 이동
4. **처음부터** 버튼으로 캔버스 초기화

## 디자인 원칙

- 군더더기 없는 미니멀 프로페셔널 디자인
- 원색 배제, 물 빠진 파스텔톤(Muted Colors)만 사용
- 학교 느린 인터넷·구형 태블릿에서도 빠르게 로딩
