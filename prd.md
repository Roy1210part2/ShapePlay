📋 제품 요구사항 정의서 (PRD): 도형놀이
1. 프로젝트 개요
•	프로젝트 이름: 도형놀이 (ShapePlay)
•	한 줄 설명: 격자 위에서 도형을 직접 만들고, 밀기/뒤집기/돌리기를 3D 입체 애니메이션으로 눈으로 보며 공간 지각력을 키우는 초등 수학 교육용 웹 프로그램.
•	프로젝트 유형: 단일 페이지 애플리케이션 (SPA - HTML/CSS/JavaScript 기반 웹).
•	개발 난이도: 중 (Medium) - 백엔드 서버는 없지만, Canvas 위에서 도형이 종이처럼 앞뒤로 뒤집히는 3D 수학적 렌더링 구현이 꽤 까다로움.
2. 사용자 시나리오
•	누가: 초등학교 4학년 학생들 및 칠판에 띄워두고 수업을 진행하는 선생님.
•	언제: 4학년 1학기 '도형의 이동' 단원 본차시 수업 시간이나 단원 평가 시.
•	왜: 머릿속으로 상상하기 힘든 도형의 회전과 뒤집기를, 뚝뚝 끊기는 그림이 아니라 부드럽게 이어지는 3D 궤적으로 직접 관찰하여 직관을 깨우치기 위해.
•	작동 흐름: 사이드바에서 [도형 만들기] 클릭 ➡️ 격자에 점을 찍어 도형 완성 ➡️ [도형 이동하기] 클릭 ➡️ 3스텝 입력창에 방향/각도/정수 입력 ➡️ 3D 플립 애니메이션과 함께 이동 ➡️ ↩️ 버튼을 눌러 과거로 되돌아가며 복기.
3. 핵심 기능 목록
•	[필수 기능]
•	20x20 캔버스 격자판 생성 및 중심점 기준 줌인/줌아웃.
•	꼭짓점을 자유롭게 이어 하나의 객체로 만드는 도형 생성 기능.
•	3스텝 이동 조작 모달: 밀기, 뒤집기, 돌리기 조건 및 정수 입력.
•	입력 에러 처리: 문자 입력 시 "숫자를 입력하세요!" 경고 표시.
•	타임라인 로그: 조작 기록을 저장하고 앞/뒤로 되돌리는 기능.
•	3D 입체 변형 애니메이션: 뒤집거나 돌릴 때 공간감을 주는 렌더링.
•	[선택 기능]
•	미션 풀어보기: 화면에 흐릿한 목표 그림자를 띄워놓고 도형을 이동시켜 정확히 겹치게 만드는 게임 모드.
4. 기술 스택
•	프론트엔드: HTML5, CSS3, Vanilla JavaScript (Canvas API). 외부의 무거운 3D 라이브러리 없이 순수 코드로 가볍게 구현.
•	배포 환경: Vercel 또는 GitHub Pages (클릭 몇 번으로 무료 배포 가능).
•	외부 연동 / AI 모델: 이 프로젝트는 서버리스(Serverless) 단독 구동 앱이므로 구글 시트 API나 AI 모델 연동은 사용하지 않습니다. 모든 데이터는 사용자의 브라우저 내부에만 안전하게 저장됩니다.
5. 화면 구성
전체 화면은 2분할 구조로 깔끔하게 구성됩니다.
•	헤더 (좌측 상단): '도형놀이' 타이틀 배치.
•	액션 버튼 (좌측 사이드바): * 타임라인: ↩️ (뒤로가기), ↪️ (앞으로가기) 아이콘.
•	메뉴: [🛠️ 도형 만들기], [🚀 도형 이동하기], [🎯 미션 풀어보기]
•	결과 섹션 (우측 메인 영역): 거대한 20x20 격자판 및 그려진 도형 표시.
•	입력 섹션 (모달 창): 이동 버튼 클릭 시 화면 중앙에 나타나는 3단계 조작 창.
•	제외 요소: 딱딱한 "AI가 생성한 텍스트입니다" 같은 안내 문구, 유치한 캐릭터, 과도한 원색 사용 전면 배제.
6. 상세 기능 명세
•	3D 플립 애니메이션 로직: 뒤집기 조작 시, 평면에서 도형을 지우고 다시 그리는 것이 아니라, JavaScript의 requestAnimationFrame과 Z축 매트릭스 변환 연산을 사용합니다. 도형의 가로폭이 서서히 줄어들었다가 다시 늘어나며 종이가 뒤집히는 듯한 깊이감을 줍니다.
•	입력 에러 처리 로직: 3스텝 모달 창의 input 필드에 keyup 이벤트를 걸어 정규식으로 검사합니다. 정수(숫자)가 아닌 값이 들어오면 즉시 팝업 창에 진동(Shake) CSS 애니메이션을 주고, 입력창 하단에 빨간 글씨로 "숫자를 입력하세요!" 텍스트를 출력합니다.
•	타임라인 배열 관리: 조작 완료 시 도형의 모든 꼭짓점 좌표 배열을 historyStack에 저장합니다. ↩️ 클릭 시 가장 최근 배열을 꺼내 캔버스를 덮어씌웁니다.
7. 디자인 가이드
•	디자인 컨셉: 군더더기 없는 미니멀리즘 프로페셔널 SaaS(소프트웨어) 느낌.
•	도형 색상 (핵심): 시각적 피로도를 낮추고 고급스러움을 더하기 위해, 원색(빨강, 파랑)을 빼고 세련된 **'물 빠진 파스텔톤(Muted Colors)'**만 사용합니다.
•	다크모드 지원: OS 시스템 설정에 반응하여 배경이 짙은 차콜색으로, 격자선이 옅은 회색으로 자동 반전되는 다크모드를 완벽 지원합니다.
8. 제약 사항
•	서버 구축 절대 금지: 선생님의 사비(유지비)가 1원도 들지 않도록 서버(Node.js, Python 등)와 DB를 구축하지 않고 브라우저 100% 로컬 환경에서 구동되도록 제한합니다.
•	무거운 라이브러리 사용 금지: 학교의 느린 인터넷 환경이나 구형 태블릿에서도 1초 만에 켜질 수 있도록 Three.js 같은 무거운 그래픽 라이브러리 사용을 금지합니다.
•	API 하드코딩 금지: 이 앱은 외부 API를 호출하지 않지만, 추후 기능이 확장되더라도 소스코드 내부에 중요 키가 노출되는 하드코딩은 배제합니다./ versel같은 걸로 무료로 가장 쉽게 배포해줘.
